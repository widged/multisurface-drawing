# multisurface-drawing

A quick experiment in drawing the same shapes across multiple surfaces (svg, canvas, webgl).


## Preview

![preview](preview.png "Preview")


## Develop

    npm run dev

## Build

    npm run bundle

## Usage

    npm run bundle
    open build/index.html
